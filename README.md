# merklizer.xyz

Web interface integrating [https://github.com/alsenet-labs/merklizer](https://github.com/alsenet-labs/merklizer)

See it in action on https://merklizer.xyz


## Build submodule Merklizer with
```
make merklizer
```

## Run and test with
```
gulp
```

## Build GitHub pages with
```
make
```
